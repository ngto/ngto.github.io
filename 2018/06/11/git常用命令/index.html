git常用命令

1、下载安装 git bash

生成公钥
$ssh-keygen -t rsa
从远程主机克隆一个版本库
$git clone <版本库地址>
$git clone <版本库地址> <本地目录名>
将远程主机更新取回本地
$git fetch
取回远程分支的更新，再与本地合并
$git pull
将本地更新推送到远程主机
$git push
查看本地分支状态
$git status
查看log日志
git log/git reflog
添加本地文件(-u表示git status列出来的上不部分更新，.表示全部更新)
$git add -u/.
$git add 文件路径
提交本地更新(不影响远程)
$git commit -m "描述"
从暂存区恢复（id可使用git log/git reflog查看）
$git reset id
比较当前文件和暂缓区文件差异(不写文件名表示比较全部)
$git diff 文件名
  • 注意:每次push之前都使用git fetch+git status+git pull 讲远程的与本地进行合并，使用git status查看没有冲突后再使用git push提交到远程。

2、分支操作

-r 查看远程分支 -a 查看所有分支
$git branch -r/-a
创建分支并切换到分支
$git checkout -b 分支名
删除分支(在master主分支删除指定的分支)
$git branch -d 分支名(git br -d name)
  • 注意：在master主分支创建分支(git checkout dev)->在分支中做修改(change dev)->修改完成添加并提交(git add -u/.+git commit -m ‘des’)->切换到master主分支并合并分支(git checkout master+git merge dev)->可删除分支(git branch -d dev)

3、记住用户名:

git config --global user.name [username]
git config --global user.email [email]

4、记住密码:

git config --global credential.helper store
